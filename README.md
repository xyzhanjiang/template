# template

Template

## Install

``` shell
git clone https://github.com/xyzhanjiang/template.git
cd template
npm install
```

or generate project from this template.

## Usage

**Develop**

``` shell
npm run start
```

Visit localhost:3000

**Build**

``` shell
npm run build
```

## Structure

``` diff
template
|- package.json
|- /public
  |- index.html  // home page
|- /src
  |- index.js    // the entry point file for the application
  |- /common
  |- /config
  |- /css        // styles
  |- /images
```
